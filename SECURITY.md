# Security Policy

## Supported Versions

Only the current `main` branch is supported.

## API Keys

Do not commit API keys, `.env.local`, shell history, request logs, or raw responses that contain private account data. Use `TWITTERAPI_IO_KEY` in your local environment or in an untracked `.env.local` file next to the installed skill.

## Reporting a Vulnerability

Please report vulnerabilities privately through GitHub Security Advisories when available. If advisories are unavailable for this repository, contact the maintainer directly rather than opening a public issue with exploit details.
